RESTful сервис для взаимодействия с колодам карт.

Автор: Качусов Степан Михайлович

Интерфейс:
1) Создать именованную колоду карт  POST api/decks/{name}
2) Удалить именованную колоду карт  DELETE api/decks/{name}
3) Получить список названий колоду  GET api/decks
4) Перетасовать колоду				GET api/decks/{name}/shuffle
5) Получить колоду по имени (в текущем состоянии)  GET api/decks/{name}

Алгоритм перетасовки задается через конфигурацию приложения, в файле appsettings.json по ключу ShuffleType:
 1) "Simple" - простой алгоритм
 2) "Manual" - эмуляцию перетасовки вручную
 
Колоды хранятся в памяти. Интерфейс IDeckRepository позволит изменить место хранения на базу данных.
(обертка)
	
Стандартная колода 52 карты. При создании отсортирована.
